# MdPhotoV v1.5.0

**A lightweight, native Windows image viewer built for speed and simplicity.**

MdPhotoV is a high-performance image viewing application developed entirely in C++ using the Win32 API and standard Windows Resources. Designed to be a zero-dependency alternative to modern bloatware, it provides a smooth, responsive experience for viewing your photos instantly.

## ✨ Key Features

- **🚀 Native Performance:** Built with pure C++ and Win32, ensuring lightning-fast startup and rendering.
- **🎨 Modern UI:** Features a clean, resource-based dialog system with a custom "About" window and native Windows styling.
- **📐 High-DPI Support:** Optimized for modern screens with crisp rendering on 4K and high-DPI displays.
- **🛡️ Zero Dependencies:** No external libraries or frameworks required. Just a single executable.
- **🔒 Modal & Stable:** Robust window management with single-instance prevention and modal child dialogs.

## 🛠️ Technology Stack

- **Language:** C++ (Modern C++17/20)
- **GUI:** Win32 API (Windows SDK)
- **Resource Management:** Native `.rc` files with Visual Resource Editor support
